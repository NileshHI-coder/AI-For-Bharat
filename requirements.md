# Smart Mentor AI – Requirements Document

 1. Project Title

Smart Mentor AI: An AI-Powered Career Guidance System for Bharat

---

 2. Problem Statement

Students from tier-2 and tier-3 cities in India face a lack of access to quality career guidance and mentorship due to high costs, limited availability of counselors, and language barriers. This results in poor career decisions, skill mismatch, and unemployment. There is a need for an affordable, scalable, AI-powered mentoring solution designed specifically for Bharat.

---

3. Objective

To build an AI-powered mentoring system that provides personalized career guidance, skill recommendations, and learning roadmaps to students, making quality mentorship accessible anytime and anywhere.

---

 4. Target Users

- School students (classes 9–12)
- College students (UG/PG)
- Students from tier-2 and tier-3 cities
- First-generation learners

---

 5. Functional Requirements

- User can  input or select from given list the particular role or job. 
- System provides personalized career recommendations
- AI suggests skill roadmaps and learning resources
- User can ask career-related questions via chat interface
- System responds with AI-generated guidance
- Admin can update career and skill data

---

6. Non-Functional Requirements

- User-friendly interface
- Low-cost and lightweight system
- Fast response time
- Scalable for large number of users
- Secure handling of user data

---

 7. System Requirements

 Hardware Requirements:
- Minimum 4 GB RAM
- Internet-enabled device (Laptop / Mobile)

 Software Requirements:

- Operating System: Windows / Linux
- Programming Language: Python
- Frontend: Streamlit
- Backend: AI / ML models
- Database: JSON / Lightweight database

---

8. Constraints

- Limited training data
- Internet dependency
- Language support initially limited to English

---

 9. Assumptions

- Users have basic digital literacy
- Internet connectivity is available
- AI responses are advisory, not professional guarantees

---

 10. Future Enhancements
 
- Multilingual support (Hindi, Gujarati, regional languages)
- Voice-based mentoring
- Offline-first access for rural areas
- Integration with government education portals
