# 🚀 Smart Mentor AI

An AI-powered career guidance system that helps students choose the right career path based on their interests, skills, and goals.

---

## 📌 Overview

Smart Mentor AI is designed to solve one of the biggest problems students face — **career confusion**.  
It provides structured career information along with AI-based guidance through an interactive chatbot.

---

## 🎯 Features

- 🎓 Career selection and detailed information  
- 🤖 AI chatbot for career guidance  
- 📊 Skills, roadmap, and salary insights  
- 🔍 Career search functionality  
- ⚡ Fast and user-friendly interface  

---

## 🧠 How It Works

- Fetches career data from local database  
- Uses AI API for intelligent responses  
- Displays personalized guidance  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Database:** JSON  
- **AI Integration:** Gemini API
